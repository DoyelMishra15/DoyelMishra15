## Selected Work

<br/>

**[CareerLens AI](https://github.com/DoyelMishra15/CareerLens-AI)**

Resume ↔ job description semantic matching engine. FastAPI backend, sentence-transformers, NLP scoring pipeline. Deployed with live metrics — `~2s/request`, `50+ resumes/min` throughput. Gap analysis included.

<br/>

**[SwarajDesk — CV Project](https://github.com/DoyelMishra15/SwarajDesk_CV_Project_)**

Vision-language model for civic grievance routing. Classifies uploaded images of infrastructure complaints and auto-routes to the correct department. Built on Vision Transformers (ViT) + TypeScript frontend. SIH 2025 National Finalist project.

<br/>

**[Sales Forecasting Dashboard](https://github.com/DoyelMishra15/Sales-Forecasting-Dashboard)**

Time-series forecasting with Meta Prophet and Scikit-learn. Streamlit interface, plug-and-play CSV upload. Model accuracy: `R² > 0.85`, `MAPE < 8%`.

<br/>

**[Employee Sentiment Analysis](https://github.com/DoyelMishra15/Employee_Sentiment_Analysis_Project)**

End-to-end NLP pipeline — raw text to actionable workplace insight. Sentiment classification, trend visualisation, modular architecture built for reuse.

<br/>

**[Event Management System](https://github.com/DoyelMishra15/EventManagementSystem)**

Full-lifecycle event management in Java. OOP design, console-driven, structured around real-world scheduling constraints.
